Markdown is a simple language that lets you write in an easy-to-read, natural format, but still allows for expressive formatting when rendered as HTML. It is used for unstructed "body text" content.

For additional reference, see GitHub's Markdown Basics.

Paragraphs

Paragraphs in Markdown are just one or more lines of consecutive text followed by one or more blank lines.

${self.example(''' On July 2, an alien mothership entered Earth's orbit and deployed several dozen saucer-shaped "destroyer" spacecraft, each 15 miles (24 km) wide. On July 3, the Black Knights, a squadron of Marine Corps F/A-18 Hornets, participated in an assault on a destroyer near the city of Los Angeles. ''')}

Headings

You can create a heading by adding one or more # symbols before your heading text. The number of # you use will determine the size of the heading.

${self.example(''' # Huge heading (<h1> tag) ## Big heading (<h2> tag) ... ###### The 6th largest heading (an <h6> tag) ''')}

Blockquotes

You can indicate blockquotes with a >.

${self.example(''' In the words of Abraham Lincoln: > Pardon my french ''')}

Styling Text

Make text bold or italic using the _ or * symbols.

${self.example(''' *This text will be italic* **This text will be bold** You can *mix* and **match** them within a line, heading, etc. You can **nest them _using_ the underscore character**. ''')}

Lists

Lists can be unordered, ordered, or nested, using the * marker or a -. ${self.example(''' * Item * Item * Item - Item - Item - Item 1. Item 1 2. Item 2 3. Item 3 1. Item 1 1. A corollary to the above item. 2. Yet another point to consider. 2. Item 2 * A corollary that does not need to be ordered. * This is indented four spaces, because it's two spaces further than the item above. * You might want to consider making a new list. 3. Item 3 ''')}

Code Formatting

Using single backticks to enclose something will format it as fixed-width text.

Note that within the backticks, other Markdown syntax will not apply.

${self.example(''' Here's an idea: why don't we take `SuperiorProject` and turn it into `**Reasonable**Project`. ''')}

A multi-line example can be enclosed with triple backticks.

${self.example(''' Check out this neat program I wrote: ``` x = 0 x = 2 + 2 what is x ``` ''')}

Links

You can create an inline link by wrapping link text in brackets ( [ ] ), and then wrapping the link in parentheses ( ( ) ).

${self.example(''' Tell all your friends to [visit Crowd Supply!](https://www.crowdsupply.com) ''')}

Images

You can insert images similarly to a link. Images can be referenced by URL (anywhere on the web).

${self.example(''' Here is a great placeholder image: ![This is a placeholder](https://placekitten.com/g/300/200) ''')}

To reference images within the Zaphod system, use a simplified URL path, like:

${self.example(''' Here is a Crowd Supply image: ![Super cool title](/img/novena-848_jpg_project-body.jpg) ''')}

Tables

Table structures can be built using the - and | symbols.

${self.example(''' Animal | Maximum Speed ----------------|-------------- Cheetah | 112-120 km/h Free-tailed bat | 96.6 km/h Pronghorn | 88.5 km/h Springbok | 88 km/h Wildebeest | 80.5 km/h Blackbuck | 80 km/h Hare | 80 km/h ''')}

When configuring images, there are three fields you can use to supply additional text metadata about the image.

Caption

The caption field is used to label an image on the page that it is associated with, and it is defined per-association, rather than per image. For example, the same image might be used a project, a tag, and an article, and it could have a different caption field for each of those uses.

It is optional, and should be a short description of what the image is showing that might be ambiguous. **Colors** should always be noted in the caption, but things like "Top View" should generally not be.

On a project/product page, the product name should not be included, as the caption will generally be shown with the context of that product.

Denim

Alt

The alt tag should only be used when the image is a **direct replacement for text**. Banners with marketing text are a good use for this. A product image that contains text (such as text written on the side of a bag or shown on the screen of a GPS unit) should not have the alt tag filled in.

The majority of images should not have an alt tag. When an alt tag is used, the title tag is optional. The alt tag is per-image, rather than per-association: it will be the same wherever the image is used.

Examples

33 Books Co.

Title

The title tag should be used as a description of the image that **stands alone from any other content**. The title tag should always be supplied, except (sometimes) when an alt tag is used.

The title tag should read as a natural description, and try to avoid arbitrary text delimiters. The title tag is per-image, rather than per-association: it will be the same wherever the image is used. Don't include any non-descriptive prefixes like "Image of".

  • Bad: "Image of SuperPod B3"
  • Bad: "Red"
  • Less Bad: "SuperPod B3 - Red"
  • Good: "SuperPod B3 Music Player in Red"
